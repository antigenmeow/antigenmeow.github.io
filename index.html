The Null Project

Negative Matters. Methods Decide. Evidence Changes.

The Null Project begins from a simple premise: negative findings matter. Scientific credit should not be reserved for positive results, statistically significant effects, or convenient narratives. A well-designed study that fails to support an expected effect can be as important as one that discovers it.

But The Null Project does not treat every negative result as evidence. A null finding earns evidentiary value only when the question is clear, the test is fair, and the conclusion is bounded by method.

The Null Project exists to build a methods-first framework for publishing, reviewing, challenging, replicating, recording, and teaching rigorous negative evidence.

Core Principles

  1. Negative Matters: Rigorous negative findings deserve the same methodological respect as positive findings.
  2. Methods Decide: The direction of a result does not determine its scientific value. The design, assumptions, precision, transparency, and interpretation do.
  3. Evidence Changes: Publication is not evidentiary finality. A finding may later be replicated, challenged, bounded, reopened, or overturned.
  4. No Null Is Doctrine: The Null Project does not turn negative findings into doctrine. Every null remains open to credible reanalysis, replication, and methodological challenge.
  5. Common Methods, Contested Evidence: We seek agreement in how claims are tested, not in what conclusions must be reached.

What Counts as a Publish-able Null

A null result is not publishable because it is negative. It is publishable because the claim was fairly tested. A negative finding is publishable only when:

  1. The scientific claim is clear and refutable.
  2. The population, exposure/intervention, comparator, outcome, and time zero are explicitly defined.
  3. The expected effect had a fair chance to appear.
  4. The study has sufficient precision to inform the claim being tested.
  5. Bias, missingness, misclassification, and confounding have been addressed or transparently bounded.
  6. The conclusion does not overstate what the data can exclude.
  7. The interpretation distinguishes absence of evidence from evidence of absence.

What The Null Project Rejects

The Null Project is not a shelter for failed studies. It is a home for fair tests. The Null Project rejects:

  • underpowered negative studies presented as evidence of no effect;
  • wide confidence intervals interpreted as absence;
  • vague hypotheses retrofitted after results are known;
  • "no significant difference" used as proof of equivalence;
  • post hoc subgroup narratives without methodological justification;
  • null findings produced by avoidable bias, poor measurement, or incoherent time zero;
  • the use of negative results as rhetorical closure.

The Null Ecosystem

  • The Null publishes methodologically rigorous negative findings.
  • The Null Reviews examines fields where evidence has stalled, failed, or been overclaimed.
  • The Null Protocol registers methods before results, including protocols designed to establish or challenge nulls.
  • The Alternative re-examines null findings without treating The Null as doctrine.
  • The Null Stands publishes replications in which the null remains supported.
  • The Null Record maintains the evidentiary status of claims over time.
  • Null Education teaches researchers to test before they claim.
  • NullPub hosts post-publication questions, concerns, replies, seminars, and status proposals.
  • Null Assembly brings these discussions into a shared annual forum.

Governance and Revision

This Constitution is a living document. It will be revised as the project develops, as methodological criticism is received, and as the community using it grows. Revisions should be public, versioned, and justified.
